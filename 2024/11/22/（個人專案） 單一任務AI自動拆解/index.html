協助 專案管理工具 來進行 WBS 拆解：

使用 Shortcut（類似Jira）管理任務項目時，很多時候一個大項目需要進行子任務拆解，這邊透過 Shortcut 提供的 API 與 ChatGPT API 進行整合，並設計 Prompt，爾後讓其拆分成子任務，具體成果如下圖。

  • 透過 Langchain 實現串接 ChatGPT API。
  • 透過 Shourtcut API 來取得對應資料。
  • 透過 Crontab 每日執行程式進行掃描。

功能展示：

  • 圖片上的 Goal 為本次任務主目標。
  • During 是執行的期間。
  • Weekdays 是平日可以進行的時間； Weekends 則是週末可進行時間。
